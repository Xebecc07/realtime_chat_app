ChatIt (Real-Time Chat App)

Made by Omkar Bordekar for CodeClause Internship

A visually enhanced, student-friendly real-time chat app. Users can join rooms, send messages instantly, and switch between light/dark mode. Built using Node.js, Express.js, and Socket.IO. Features chat sound, bubbles, and a clean UI.

How to Run:
1. Install Node.js: https://nodejs.org/
2. Open terminal in the project folder
3. Run: npm install express socket.io
4. Start server: node server.js
5. Open http://localhost:3000 in browser

Open multiple tabs or devices to chat live across rooms.
